## Getting Started
### Requirements
* Intellij IDEA
* Java

Library for dependencies on build.gradle:
* Selenium 3.141.59
* Cucumber 5.6.0
* JUnit 4.12

### Usage
1. Clone this project
2. Open the project on Intellij IDEA
3. Run CucumberRunner
